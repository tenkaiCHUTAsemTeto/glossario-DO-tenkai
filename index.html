    Glossário de Termos de Programação

    Glossário de Termos de Programação

    Grupo 1

    HTML
    Definição: Linguagem de marcação utilizada para estruturar conteúdo na web, definindo elementos como textos, imagens, links e outros componentes de uma página.
    Sinônimos:
    • Linguagem de marcação
    • Código de estruturação
    • Formato web
    Implementação
    Definição: Ato de colocar em prática ou executar um plano, sistema ou código, especialmente em programação, onde se refere à criação ou aplicação de funcionalidades.
    Sinônimos:
    • Execução
    • Realização
    • Aplicação
    Undefined
    Definição: Em programação, termo usado para indicar que uma variável ou objeto não possui um valor definido ou atribuído, comum em linguagens como JavaScript.
    Sinônimos:
    • Indefinido
    • Não definido
    • Vazio
    JavaScript
    Definição: Linguagem de programação de alto nível usada principalmente para adicionar interatividade e dinamismo a páginas web.
    Sinônimos:
    • JS
    • Script web
    • Código dinâmico
    Desenvolvendo
    Definição: Ato de criar, construir ou aprimorar algo, como um software, aplicação ou funcionalidade, por meio de programação ou design.
    Sinônimos:
    • Criando
    • Construindo
    • Elaborando

    Grupo 2

    Parâmetro
    Definição: Em programação, uma variável ou valor passado para uma função ou método que define como a função deve operar.
    Sinônimos:
    • Argumento
    • Variável de entrada
    • Configuração
    Estacionado
    Definição: No contexto de tecnologia, pode se referir a um projeto, domínio ou recurso temporariamente inativo ou mantido em espera.
    Sinônimos:
    • Parado
    • Suspenso
    • Inativo
    Função (em código de programação)
    Definição: Bloco de código projetado para executar uma tarefa específica, que pode ser chamado em diferentes partes de um programa.
    Sinônimos:
    • Método
    • Rotina
    • Procedimento
    Cobrado
    Definição: No contexto de tecnologia, pode se referir a um serviço, recurso ou ação que exige pagamento ou uso de recursos computacionais.
    Sinônimos:
    • Taxado
    • Faturado
    • Debitado
    Completa
    Definição: Algo finalizado, inteiro ou que atingiu seu estado final, como uma tarefa, código ou processo em programação.
    Sinônimos:
    • Concluída
    • Finalizada
    • Íntegra

    Grupo 3

    Implementa
    Definição: Ato de colocar em ação ou executar uma funcionalidade, sistema ou código em um programa ou aplicação.
    Sinônimos:
    • Executa
    • Aplica
    • Realiza
    addEventListener
    Definição: Método em JavaScript usado para associar uma função a um evento específico (como clique ou tecla pressionada) em um elemento HTML.
    Sinônimos:
    • Vincular evento
    • Escutar evento
    • Associar evento
    Identifique
    Definição: Ato de reconhecer, determinar ou localizar algo, como uma variável, elemento ou erro em um programa.
    Sinônimos:
    • Reconheça
    • Localize
    • Determine
    Contador
    Definição: Em programação, uma variável ou mecanismo usado para contar ocorrências, iterações ou eventos.
    Sinônimos:
    • Incrementador
    • Totalizador
    • Marcador
    Text Content
    Definição: Propriedade em JavaScript que define ou retorna o conteúdo de texto de um elemento HTML, sem incluir as tags.
    Sinônimos:
    • Conteúdo de texto
    • TEXTO interno
    • Valor de texto
